# SMA ADP API (Golang)

Backend inisialisasi untuk migrasi dari NestJS ke Golang (Phase 0). Jalankan dev env via Docker.

## Quickstart
```bash
cp .env.example .env
make docker-up
make dev
```

## Docs
- Swagger: `/docs` (dev only)
- Health: `/health`, `/ready`

## Makefile
Lihat target via `make help`.
